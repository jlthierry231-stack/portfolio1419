# portfolio1419
Personal Developer Portfolio

This repo contains a small single-page portfolio for Jeremy Thierry with three interactive demo projects:

- `projects/dashboard` — Animated Buffalo Wings data dashboard (Chart.js demo)
- `projects/task-manager` — Smart task manager with localStorage
- `projects/battleship` — Full Battleship game with ship placement and AI opponent

To view locally, run a static file server in the repo root and open http://localhost:8000/index.html

Quick serve commands:

PowerShell / Windows (from repo root):
```powershell
python -m http.server 8000
```

Or with Node.js installed:
```bash
npx http-server -p 8000
```
